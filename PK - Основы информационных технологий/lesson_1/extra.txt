RAM memory

Random Access Memory (RAM) is a computer's short-term memory,
where data that the processor is currently using is stored temprorarily. 

Случайно-доступная память. Это кратковременная компьютерная память,
где хранятся данные которые используются процессором сейчас.

RAM memory can be accessed much faster than data on a hard disk, SDD, 
or other long-term storage device, which is why RAM 
capacity is so important for system perfomance.

RAM может быть доступна намного быстрее чем данные на жестком диске,
или другом долгосрочном хранилище, поетому обьем РАМ так важен для 
системной производительности.

Random-access memory RAM is a form of computer memory
that ca be read and changed in any order, typically used to
store working data and machine code.

РАМ память это тип компьютерной памяти который может быть 
изменемн в любом порядке, обычно используется для 
хранения рабочих (активных) данных и машиного кода

Существуют два основных типа оперативной памяти:

Один из них называется Static RAM (Static RAM / SRAM). STAM очень быстрый и в настоящее время
является самым быстрым запоминающим устройством для чтения и записи, но также очень дорого,
поэтому используется только в местах с очень высокими требованиями, например, 
на первом уровне ЦП. Буфер, вторичный буфер.

Другой называется димнамической RAM (Dynamic RAM / DRAM). DRAM сохраняет данные в течение
короткого времени, а скорость ниже, чем у SRAM, но все же быстрее, чем любое ROM, но DRAM дешевле, 
чем SRAM с точки зрения цены. Во многих случаях компьютерная память - это DRAM.

SRAM: статическая память с произвольным доступом, которая должна обновлять данные время от времени.
Однако его уровень интеграции относительно низок, и он не подходит для памяти большой емкости,
которая обычно используется в кэш-памяти процессора.
Процессоры ARM9, такие как S3C2330, имеют 4КБ SRAM для запуска ЦП.

SDRAM: синхронная динамическая память с произвольным доступом, такая как компьютерная память,
такая оперативная память называется DDR SDRAM. Его уровень интеграции очень высок. Поскольку он
динамичен, должна быть схема обновления, и данные должны обновляться через определенные промежутки времени.
Единицы хранения не расположены линейно, а разбиты на страницы. Память во встроенных продуктах общего назначения - SDRAM.

Память используется для хранения данных и программ, которые в настоящее время используются (то есть выполняются).
Компьютерная память, которую мы обычно так назывем, относится к димнамической памяти (DRAM). Так называемая <<динамическая>> 
в динамической памяти относится к когда мы записываем данные в ВКФЬб по прошествии некоторого времени данные будут потеряны, 
поэтому для обновления памяти требуется дополнительная схема.

Конкретный рабочий процесс выглядит следующим образом: хранит ли ячейка памяти DRAM 0 или 1,
зависит от того, имеет ли конденсатор заряд, причем заряд равен 1, а нулевой заряд равен 0. 
Но в течение длительного периода времени конденсатор, представляющий 1, будет разряжаться, а конденсатор,
представляющий 0, будет поглощать заряд. Это причина потери данных. Операция обновления периодически проверяет конденсатор.
Если аккумулятор составляет более 1/2 от полной батареи, считается, что он соответсвует 1. , И полностью зарядите конденсатор; если мощность меньше 1/2,
считается, что он представляет собой 0, и конденсатор разряжатся для поддержания неприрывности данных.

ПЗУ: общий термин для постоянной памяти.

PROM: Программируемая постоянная память может быть записана только один раз. Если она написана неправильно, она будет списана.
